Get started

Three commands to production.

# Install the Helix CLI
npm i -g @helix/cli

# Deploy your first model
helix deploy llama-3.1-70b

# Start calling it
curl https://api.helix.ai/v1/chat \
  -H "Authorization: Bearer $HELIX_KEY" \
  -d '{"model":"llama-3.1-70b","messages":[{"role":"user","content":"Hello"}]}'
