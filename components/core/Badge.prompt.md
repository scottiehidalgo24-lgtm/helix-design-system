Status / category pill, set in mono. Use the dot for live status (running, healthy).

```jsx
<Badge tone="green" dot>Running</Badge>
<Badge tone="warning" variant="soft">Scaling</Badge>
<Badge tone="neutral" variant="outline">v3</Badge>
<Badge tone="danger" dot>Failed</Badge>
```

- `tone`: neutral / green / success / warning / danger / info.
- `variant`: `soft` (default, translucent) · `solid` · `outline`.
- Keep labels to one or two words.
